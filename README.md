# 🎓 KiddiLearn

A web-based **Preschool Learning Management System** developed as a capstone project to support preschool education through interactive learning and effective communication between teachers and parents.

---

## 📖 Project Overview

KiddiLearn is a web-based learning management system designed for preschool education. The system provides teachers with tools to manage learning materials, worksheets, student activities, and progress, while allowing parents to monitor their child's learning journey through a dedicated dashboard.

The project aims to provide an engaging and child-friendly learning environment by combining interactive educational content with an intuitive and responsive user interface.

---

## 🎯 Objectives

* Provide an interactive learning platform for preschool students.
* Enable teachers to manage lessons, worksheets, and student activities.
* Allow parents to monitor their child's learning progress.
* Improve communication between teachers and parents.
* Create a responsive and child-friendly learning experience.

## 🚀 Installation

1. Clone this repository.
2. Place the project inside your XAMPP `htdocs` directory.
3. Start **Apache** and **MySQL** in XAMPP.
4. Import `database/kiddielearn.sql` into phpMyAdmin.
5. Update the database configuration in `db.php` if necessary.
6. Open your browser and visit:

```
http://localhost/kiddielearn
```
